# X Library Feed

Public RSS, Markdown, JSON, and daily archive snapshots generated from two X accounts' bookmarks and favorites.

- RSS: `rss.xml`
- Latest Markdown: `public/x-bookmarks-favorites.md`
- Latest JSON: `public/x-bookmarks-favorites.json`
- Direct image URLs: `image_urls` in JSON and `media:content` in RSS
- Archive: `archive/x-bookmarks-favorites/YYYY-MM-DD.md`

The GitHub Action runs hourly.
